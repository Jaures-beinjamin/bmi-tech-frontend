This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

---

## 🌟 Présentation de la startup
**BMI-Tech** est une startup suisse innovante spécialisée dans la **robotique et l’informatique appliquée**.  
Le site **bmi-tech.ch** présente le projet phare de la startup : un **robot joueur d’échecs moderne**, inspiré du célèbre **Turc mécanique du XVIIIe siècle**, développé par l’inventeur **Roland Burgermeister**.

Le projet allie **vision artificielle, programmation avancée et mécanique fine**, avec pour ambition de créer un robot capable de **voir les pièces sur l’échiquier, les déplacer et reproduire les gestes humains**.

---

## 🧩 Objectifs du site
- Présenter la **startup et son inventeur**  
- Valoriser le **projet robotique et ses innovations techniques**  
- Informer sur les **technologies utilisées**  
- Permettre aux visiteurs de **contacter la startup** pour partenariats ou collaborations  

---

## ⚙️ Stack technique

### Frontend
- **Next.js** (framework React moderne avec SSR pour SEO)  
- **React** pour les composants interactifs  
- **Tailwind CSS / Fluent UI** pour le design et la réactivité  

### Backend / API
- **ASP.NET Core** pour les endpoints API (formulaires, gestion des données)  
- Sécurisation des requêtes et logique métier  

### Base de données
- **Azure SQL Database** pour données relationnelles  
- **Cosmos DB** (optionnel) pour données NoSQL ou démo robot  

### Hébergement / Cloud
- **Microsoft Azure** pour hébergement complet : Frontend, API, base de données, domaine `bmi-tech.ch`
